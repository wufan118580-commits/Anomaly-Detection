1.检查SSH密钥配置
# 查看是否已有 SSH 密钥
ls -la ~/.ssh

# 如果没有，生成新的 SSH 密钥
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

# 查看公钥并添加到 GitHub
cat ~/.ssh/id_rsa.pub

然后将公钥添加到 GitHub：

登录 GitHub

Settings → SSH and GPG keys → New SSH key

粘贴公钥内容

2.测试 SSH 连接
ssh -T git@github.com

如果配置正确，会显示：Hi username! You've successfully authenticated...

3. 使用 HTTPS 替代 SSH
git clone https://github.com/wufan118580-commits/ai-learn.git

4.检查网络连接
# 测试 GitHub 连通性
ping github.com
# 或
curl -v https://github.com

5. 配置 SSH 客户端
# 配置 SSH 使用 443 端口（绕过防火墙）
echo "Host github.com
  Hostname ssh.github.com
  Port 443" >> ~/.ssh/config

# 测试 SSH 连接
ssh -T git@github.com

6.手动添加remote
# 添加 remote
git remote add origin https://github.com/wufan118580-commits/ai-learn.git

# 或者如果是 SSH
git remote add origin git@github.com:wufan118580-commits/ai-learn.git

# 验证
git remote -v
